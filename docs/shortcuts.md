# Keyboard shortcuts

## Tool shortcuts

| Key | Tool |
|---|---|
| V | Select |
| R | Rectangle |
| E | Ellipse |
| A | Arrow |
| L | Line |
| T | Text |
| N | Numbered marker |
| P | Freehand pen |
| H | Highlight |
| B | Blur |
| C | Crop |

## Editor shortcuts

| Shortcut | Action |
|---|---|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+S | Save |
| Ctrl+Shift+S | Save as |
| Ctrl+C | Copy to clipboard |
| Ctrl+D | Clone selected |
| Ctrl+A | Select all |
| Delete | Delete selected |
| Ctrl+W | Close editor |
| Ctrl+0 | Fit to window |
| Ctrl++ | Zoom in |
| Ctrl+- | Zoom out |

## Capture keybindings

These are set up by `waysnip setup`:

| Key | Action |
|---|---|
| PrintScreen | Region capture |
| Ctrl+PrintScreen | Fullscreen capture |

## Region selector

| Key | Action |
|---|---|
| Click + drag | Draw selection |
| Drag edges/corners | Adjust selection |
| Enter | Confirm selection |
| Escape | Cancel |
