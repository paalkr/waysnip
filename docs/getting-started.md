# Getting started

## First capture

Press **PrintScreen** or run:

```bash
waysnip region
```

## The region selector

The screen freezes and you get a crosshair cursor with a magnifier.

- Click and drag to draw a selection box
- Drag the edges or corners to adjust
- The magnifier shows a zoomed view around the cursor for pixel-precise selection
- Press **Enter** to confirm, **Escape** to cancel

## The editor

After confirming the region, the editor opens with your screenshot.

- The toolbar at the top has all annotation tools (rectangle, arrow, text, blur, etc.)
- Pick a tool and draw on the image
- Use the color picker and width controls to style your annotations
- **Ctrl+Z** to undo, **Ctrl+Y** to redo
- **Ctrl+S** to save, **Ctrl+C** to copy to clipboard

Annotations are saved inside the PNG file. When you open the image later, you can still move and edit every annotation.

## Gallery

Open the gallery to browse saved screenshots:

```bash
waysnip gallery
```

- Double-click any screenshot to re-open it in the editor
- Right-click for actions: copy, edit, reveal in file manager, flatten, delete
- The gallery updates live when new screenshots are saved
